# 视角：追求节奏（Chase Rhythm）

> 被 prompts/user_analyzer.md 调用。
> 目标：识别此人在关系中的追求强度、撤退策略和热度维持方式。

---

## 三个核心维度

### 1. 追求强度（Pursuit Intensity）

读取信号：
- 主动联系的频率（每天几次？对方不回应后是否继续）
- 话题发起的比例（谁更多在开话头）
- 情感投入的速度（升温快还是慢）
- 在对方冷淡时的反应（加倍追还是等待）

```
追求强度：{低 / 中 / 高 / 极高}
表现：{描述}
```

### 2. 撤退策略（Withdrawal Strategy）

读取信号：
- 是否存在主动冷却行为（突然减少联系、已读不回、晾一段时间）
- 撤退的触发条件（被拒绝后？对方太主动时？测试对方反应？）
- 撤退的持续时长和回归方式
- 撤退是有意为之还是情绪驱动

```
撤退策略：{无 / 被动撤退 / 主动撤退}
触发条件：{描述}
持续模式：{描述}
```

### 3. 热度维持（Heat Management）

读取信号：
- 关系进入平稳期后的维系行为
- 是否制造稀缺感（不总是立即回复、偶尔消失）
- 是否维持多线（同时与多人保持暧昧或联系）
- 话题掌控能力（能否持续制造新鲜感）

```
热度维持方式：{描述}
稀缺感制造：{有意 / 无意 / 无}
多线迹象：{有 / 无 / 不确定}
```

---

## 综合输出

```
追求节奏总结：{2–3句描述此人在关系中的整体追求行为模式}
与舔狗/海王谱的对应：{偏向谱的哪个方向，基于以上三个维度}
```
