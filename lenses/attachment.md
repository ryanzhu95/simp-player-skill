# 视角：依恋模式（Attachment）

> 被 prompts/user_analyzer.md 调用。
> 目标：从对话数据中识别此人的依恋倾向，不预设结论。

---

## 读取信号

从对话中寻找以下信号，记录原文依据：

**焦虑型信号**
- 对方未及时回复时的反应（催促、自我否定、过度解释）
- 主动联系的频率和触发条件
- 在冲突中快速道歉或撤回立场
- 寻求反复确认（"你还喜欢我吗"类表达）

**回避型信号**
- 情感话题的转移或敷衍
- 忙碌/消失后的解释方式（或不解释）
- 对亲密感的推拒或降温
- 主动联系少，但回应质量高

**安全型信号**
- 冲突后能平静表达需求
- 不在意对方回复速度
- 能接受对方有独立空间

**混乱型信号**
- 同一人身上同时出现焦虑和回避信号
- 亲密时推开，疏远时追回

---

## 输出格式

```
依恋倾向：{类型}（焦虑型 / 回避型 / 安全型 / 混乱型）
置信度：{高 / 中 / 低}（样本少于50条时标注"样本偏少"）
核心表现：{2–3条原文依据或行为描述}
注意：{若存在矛盾信号，在此说明}
```
