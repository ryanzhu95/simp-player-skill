# 视角：冲突风格（Conflict）

> 被 prompts/user_analyzer.md 调用。
> 目标：从对话数据中识别此人在关系冲突中的处理惯性。

---

## 读取信号

**顺从型**
- 快速认错，即使并非明显过错方
- 用"算了"、"我不介意"结束争论
- 冲突后主动缓和，不追究

**回避型**
- 沉默、已读不回、转移话题
- 用忙碌或外部事件作为退出理由
- 冲突过后装作没发生

**竞争型**
- 坚持己方立场，不轻易让步
- 反问、举例、拉长争论
- 冲突中情绪升级而非降级

**合作型**
- 主动询问对方感受
- 表达"我们一起解决"的意图
- 冲突后有复盘或总结

**妥协型**
- 各退一步，快速达成临时协议
- 不深究根本原因

---

## 特别关注

- **冲突后的追责行为**：是否事后翻旧账？
- **道歉的质量**：是真实认错还是为了终止冲突？
- **触发阈值**：什么程度的摩擦才会引发冲突反应？

---

## 输出格式

```
冲突风格：{主要类型}（可标注混合，如"顺从+回避"）
触发阈值：{低 / 中 / 高}
道歉模式：{主动 / 被动 / 极少}
核心表现：{2–3条原文依据或行为描述}
```
